# shi-jian

***该库仍在开发中***

shi-jian，即时间，一个富有人体工学的现代日期库

## 安装

`npm install shi-jian`

`pnpm add shi-jian`

`bun add shi-jian`

## 注意事项

- shi-jian的主要功能在于解析与格式化日期对象，目前暂不考虑helpers实现

- shi-jian当前的测试内容极其不完善
